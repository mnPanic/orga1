## Circuitos

Para el armado de los circuitos se usa [Logisim](http://www.cburch.com/logisim), como es recomendado por la guia.

### Convencion de nombres para circuitos

P.E.I

- **P**: Practica
- **E**: Ejercicio
- **I**: Item

*Ej: 2.5.a = Practica 2, Ejercicio 5, Item a*
